# 🔍 OSINT Reconnaissance Tool

A Python-based OSINT tool for gathering public intelligence on domains.

## Features
- 🔎 WHOIS Lookup — find who owns a domain
- 🌍 IP & Geolocation Lookup — get location, ISP, timezone
- 📧 Email Scraper — extract emails from any webpage
- 📄 Page Metadata Extractor — titles, links, descriptions

## Installation
pip install requests beautifulsoup4 python-whois colorama

## Usage
python osint.py
